# bulletr 📓

**bulletr** 是一款基于 Flutter 开发的极简主义子弹笔记 (Bullet Journal) 应用。它结合了传统手帐的灵活性和数字化的便利性，帮助你高效管理日程、任务和灵感。

> 💡 **设计理念**：从宏观（年/月）到微观（周/日）的无缝流转，通过拖拽轻松规划你的人生。

## ✨ 核心功能 (Features)

### 1. 多维度时间视图 📅
- **日视图 (Daily)**：极简列表，支持完成、编辑、删除任务。
- **周视图 (Weekly)**：顶部任务池 + 每日列表，支持将周任务拖拽分配到具体日期。
- **月视图 (Monthly)**：全月概览，支持将月度目标拖拽分配到具体日期。
- **年视图 (Yearly)**：年度规划，支持将年度宏愿拖拽分配到具体月份。

### 2. 强大的拖拽交互 (Drag & Drop) 👆
- **长按排序**：在日视图和集子中，长按即可调整任务顺序。
- **跨维度规划**：
    - 从“年”拖到“月”。
    - 从“月”拖到“日”。
    - 从“周”拖到“日”。
- **反向归档**：支持将任务从具体日期拖回顶部的“任务池”，重新变回周/月/年度目标。

### 3. 子弹笔记体系 (Bujo Keys) 🔑
- **• 任务 (Task)**：支持点击打勾完成 (实心黑点/删除线)。
- **○ 事件 (Event)**：记录发生的客观事实。
- **- 笔记 (Note)**：记录想法或灵感。

### 4. 集子与收集箱 (Collections) 📂
- **收集箱 (Inbox)**：快速捕捉无日期的想法。
- **自定义集子**：创建专属分类（如“书单”、“旅行计划”），任务既可以属于集子，也可以同时拥有日期。
- **侧边栏管理**：通过左侧抽屉式菜单快速切换和管理集子。

### 5. 极致体验 🎨
- **侧滑菜单**：左滑任务可快速修改日期或删除。
- **全功能编辑**：点击任务可修改内容、类型、日期及归属集子。
- **高级日期选择器**：支持日、周、月、年四个维度的快速选择。

## 🛠 技术栈 (Tech Stack)

本项目使用 **Flutter** 构建，主要使用了以下技术和库：

- **UI Framework**: Flutter (Material 3 Design)
- **State Management**: `provider` (轻量级状态管理)
- **Internationalization**: `intl` (日期格式化)
- **UI Components**: 
  - `flutter_slidable` (侧滑菜单)
  - `ReorderableListView` (拖拽排序)
- **Utilities**: 
  - `uuid` (生成唯一ID)
  - `url_launcher` (跳转网页/邮件)

## 🚀 快速开始 (Getting Started)

如果你想在本地运行此项目：

1.  **克隆仓库**
    ```bash
    git clone https://github.com/你的用户名/bulletr.git
    cd bulletr
    ```

2.  **安装依赖**
    ```bash
    flutter pub get
    ```

3.  **运行项目**
    ```bash
    flutter run
    ```
